== Description ==
WPO Checker allows to you to check performance of  your posts and terms. You'll test your website contents  with Gtmetrix, Google PageSpeed Insights, Web Page Test, Page Speed Optimization, Yellow Lab Tools and Batch Speed.

== Installation ==
1. Upload `wpo-checker` to the `/wp-content/plugins/` directory.
1. Activate the plugin through the `Plugins` menu in WordPress.
1. Go to Settings, `WPO Checker` menu
1. Select one or more wpo checker sites.
1. Go to posts list or terms list in wp-admin zone and click in quick link( links below title ) of a wpo checker site.
