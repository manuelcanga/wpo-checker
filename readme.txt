== Description ==
WPO Checker allow to you to check performance of  your posts and terms. You\'ll test your website contents  with Gtmetrix, Google PageSpeed Insights, Web Page Test, Page Speed Optimization, Yellow Lab Tools and Batch Speed.

= Send me bugs or improvements

If you’re interested in contributing to WPO Checker we’ve got more than 350 contributors, and there’s always room for more. Head to the [WPO Checker GitHub Repository](https://github.com/trasweb/wpo-checker/).


== Installation ==
1. Upload \"wpo-checker\" to the \"/wp-content/plugins/\" directory.
1. Activate the plugin through the \"Plugins\" menu in WordPress.
1. Go to Settings, WPO Checker menu
1. Select one or more wpo checker sites.
1. Go to posts list or terms list in click in quick link of a wpo checker site.
