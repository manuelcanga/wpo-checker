== Description ==
WPO Checker allows to you to check performance of  your posts and terms. You'll test your website contents  with Gtmetrix, Google PageSpeed Insights, Web Page Test, Page Speed Optimization, Yellow Lab Tools,  Batch Speed, Keycdn Performance Test and Bytecheck.

== Installation ==
1. Upload `wpo-checker` to the `/wp-content/plugins/` directory.
1. Activate the plugin through the `Plugins` menu in WordPress.
1. Go to Settings, [`WPO Checker` menu](https://ps.w.org/wpo-checker/assets/screenshot-1.png?rev=2218093)
1. Select one or more wpo checker sites.
1. Go to [posts list](https://ps.w.org/wpo-checker/assets/screenshot-2.png?rev=2218093) or [terms list](https://ps.w.org/wpo-checker/assets/screenshot-3.png?rev=2218093) in wp-admin zone and click in quick link( links below title ) of a wpo checker site.
1. Look at [screenshots](https://wordpress.org/plugins/wpo-checker/#screenshots) in order to get visual information.
